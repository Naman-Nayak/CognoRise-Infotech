# Breast Cancer Classification

## Overview

This project aims to classify breast cancer tumors as Malignant (Cancerous) or Benign (Non-Cancerous) using Machine Learning algorithms.

## Dataset

Breast Cancer Wisconsin Dataset

## Algorithms Used

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree Classifier
* Random Forest Classifier

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Train-Test Split
6. Feature Scaling
7. Model Training
8. Model Evaluation

## Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 97.37%   |
| Random Forest       | 96.49%   |
| KNN                 | 94.73%   |
| Decision Tree       | 94.73%   |

## Conclusion

Among all the models, Logistic Regression achieved the highest accuracy of 97.37% and performed best for breast cancer classification.
