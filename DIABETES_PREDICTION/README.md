# Diabetes Prediction

## Overview

This project aims to predict whether a person has diabetes based on medical and lifestyle-related features using Machine Learning classification algorithms. The model analyzes health-related parameters such as age, BMI, blood glucose level, HbA1c level, hypertension, heart disease, and smoking history to determine diabetes risk.

## Dataset

Diabetes Prediction Dataset

## Algorithms Used

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier
- Random Forest Classifier
- AdaBoost Classifier

## Results

| Model | Accuracy |
|--------|----------|
| AdaBoost Classifier | 97.22% |
| Random Forest Classifier | 97.08% |
| K-Nearest Neighbors (KNN) | 96.13% |
| Logistic Regression | 95.87% |
| Decision Tree Classifier | 95.24% |

## Conclusion

Among all the classification models, AdaBoost Classifier achieved the highest accuracy of **97.22%** and performed best for diabetes prediction. This project demonstrates how machine learning can support early disease prediction using healthcare-related data.
