# House Price Prediction

## Overview

This project aims to predict house prices using Machine Learning regression algorithms based on various housing features such as bedrooms, bathrooms, living area, lot size, house condition, and construction details.

## Dataset

House Price Prediction Dataset

## Algorithms Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor
* Gradient Boosting Regressor

## Project Workflow

1. Data Loading
2. Data Cleaning & Preprocessing
3. Handling Missing Values
4. Outlier Detection & Removal
5. Exploratory Data Analysis (EDA)
6. Feature Selection
7. Train-Test Split
8. Model Training
9. Model Evaluation
10. Performance Comparison

## Results

| Model                       | R² Score |
| --------------------------- | -------- |
| Gradient Boosting Regressor | 0.527    |
| Random Forest Regressor     | 0.502    |
| Linear Regression           | 0.493    |
| Decision Tree Regressor     | 0.046    |

## Conclusion

Among all the regression models, Gradient Boosting Regressor achieved the highest R² score of **0.527** and performed best for predicting house prices. The project demonstrates how machine learning regression techniques can be used to estimate property prices based on housing characteristics.
